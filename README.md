# :blue_heart: Weather-Metrika / Vite / Tailwind 4.0

приложение по отслеживанию погоды и прогнозов на ближайшее время по запросу

### Основной стек: 
- Vite
- Tailwind
- Recharts
- Tanstack/React-query
- Axios
- Jotai
